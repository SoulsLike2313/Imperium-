# MANUAL PROOFS — AGENT MAP

Path:
E:\IMPERIUM\ARTIFACTS\_MANUAL_PROOFS

Role:
This folder stores manually operated proof artifacts. These are records of actions performed by the Owner or a human operator using terminal commands, before the workflow becomes a reusable script or automated tool.

What lives here:
- manual SSH proofs
- manual prompt dispatch proofs
- manual fetch/open/copy proofs
- human-operated smoke tests
- manual route validation evidence
- proof that later becomes a Python tool or Sanctum button candidate

Allowed writes:
- manually verified proof task folders
- receipts from human-operated commands
- redacted command notes
- source pointers to SSH_COMMAND_LIBRARY recipes
- owner summaries

Forbidden writes:
- raw private keys
- passwords
- tokens
- .env files
- uncontrolled local configs
- automatic watcher outputs
- canonical THRONE material

Policy:
Manual proof is not automation.
Manual proof is not canon.
Manual proof is evidence that a command sequence worked once under human control.
A manual proof may become a Python tool only after review and smoke-test.
