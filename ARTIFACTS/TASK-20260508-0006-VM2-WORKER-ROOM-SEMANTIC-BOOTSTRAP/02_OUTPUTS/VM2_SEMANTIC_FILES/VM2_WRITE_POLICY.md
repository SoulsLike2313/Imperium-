# VM2 WRITE POLICY

- VM2 writes only under ~/IMPERIUM_WORKER_ROOM.
- VM2 does not write to THRONE.
- VM2 does not write to PC directly unless a specific route task allows it.
- VM2 does not delete.
- VM2 does not auto-sync.
- VM2 outputs must be fetched by PC or explicitly dispatched through an approved route.
