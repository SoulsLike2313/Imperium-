# FORTY STAGE LARGE TASK DESIGN EXAMPLE

This is a non-executable design pattern for large decomposition.

- 40 stages grouped into 8 phase groups.
- each phase has explicit entry gate and sync point.
- contour distribution example: PC(16), VM2(16), OWNER_MANUAL(8 decision/control stages).
- every stage defines required tool_ids and pass criteria.
- every phase end requires Inquisition checkpoint.

Purpose:
Scale planning model only, not implementation.
