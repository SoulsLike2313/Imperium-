# IMPERIUM ROOT AGENT MAP

ARCHIVE: зона хранения legacy/old материалов и исторических импортов из root.
ARTIFACTS: зона task-артефактов по схеме E:\IMPERIUM\ARTIFACTS\<TASK_ID>\...
OBSERVED: зона observed-копий и исторических snapshot-поверхностей.
SSH_COMMAND_LIBRARY: библиотека SSH маршрутов, инструментов и инструкций dispatch/fetch.
PC_ENGINEERING_ROOM: активная инженерная зона анализа, планов и рабочих карт.

В root нельзя класть task outputs, zip/sha256/reports/maps.
Новые outputs класть только в E:\IMPERIUM\ARTIFACTS\<TASK_ID>\...
