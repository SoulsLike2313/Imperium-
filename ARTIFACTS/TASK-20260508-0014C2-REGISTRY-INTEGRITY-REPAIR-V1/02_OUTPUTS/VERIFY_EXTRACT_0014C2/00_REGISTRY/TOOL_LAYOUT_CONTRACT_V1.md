# TOOL LAYOUT CONTRACT V1

An active script is active only if:
- it is in the correct class folder;
- it is listed in TOOLS_MASTER_INDEX.json;
- it has a tool_id;
- it has sha256;
- it has purpose and input/output contract;
- it declares receipt/ledger/provenance/owner-report behavior;
- it is not only a stray file;
- it does not use latest-bundle logic;
- it does not perform THRONE/watchers/automation actions.
