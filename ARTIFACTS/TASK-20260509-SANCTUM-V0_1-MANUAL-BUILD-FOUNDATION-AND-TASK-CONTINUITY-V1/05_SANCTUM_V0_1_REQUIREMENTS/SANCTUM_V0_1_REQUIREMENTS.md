# SANCTUM V0.1 REQUIREMENTS

Sanctum v0.1 in this route is a local control shell for manual pipeline work.

Required features:
1. Open Astra Utility.
2. Open Explorer.
3. Show task list from `E:\IMPERIUM\ORGANS\ASTRONOMICON\TASKS`.
4. Show selected task detail panel.
5. Show `STAGE_MAP.json` if present.
6. Show `NEXT_ALLOWED_ACTION.json` if present.
7. Show `ROUTE_STATUS.json` if present.
8. Provide manual notes area.
9. Open selected task folder.
10. Refresh task list.

Scope boundary: manual and local-only, no autonomous execution behavior.
