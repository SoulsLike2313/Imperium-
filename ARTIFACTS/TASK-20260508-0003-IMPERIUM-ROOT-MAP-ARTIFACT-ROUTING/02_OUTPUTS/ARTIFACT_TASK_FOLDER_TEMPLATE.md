# ARTIFACT TASK FOLDER TEMPLATE

Для каждой задачи создавать:

E:\IMPERIUM\ARTIFACTS\<TASK_ID>\

Структура:

<TASK_ID>\
├── 00_AGENT_MAP.md
├── 01_INPUTS\
├── 02_OUTPUTS\
├── 03_RECEIPTS\
├── 04_MANIFESTS\
├── 05_HASHES\
├── 06_BUNDLES\
├── 07_REPORTS\
├── 08_OWNER_SUMMARY\
├── 09_SOURCE_POINTERS\
└── FINAL_STEP_BUNDLE\

00_AGENT_MAP.md внутри task folder должен содержать:

- task_id
- task name
- purpose
- inputs
- outputs
- receipts
- bundles
- current status
- source pointers
- owner notes
- next action
