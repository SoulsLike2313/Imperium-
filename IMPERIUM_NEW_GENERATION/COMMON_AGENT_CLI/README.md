# COMMON_AGENT_CLI Reference V0.1

COMMON_AGENT_CLI is the shared terminal UX foundation for IMPERIUM local organ agents.

Goals:
- product-grade operator shell;
- mode discipline and short slash commands;
- renderer truth (rich vs ansi fallback);
- honest visual evidence;
- original IMPERIUM heraldry and color identity.

This layer is reference-first and reusable by Officio and future organ agents.
No external logo, no Kilo dependency, no copied assets.
