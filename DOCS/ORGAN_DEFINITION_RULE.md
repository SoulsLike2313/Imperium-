# ORGAN DEFINITION RULE

An IMPERIUM organ is not operational unless all mandatory definition fields exist and are verifiable.

## Mandatory Definition Fields

- doctrine/card
- ports
- schemas
- scripts/tools or declared script interface
- receipts
- personal utility/dashboard requirement
- Sanctum visibility requirement
- maturity level
- task lifecycle role
- current limitations

## Maturity Levels

- `LEVEL 0 NAME_ONLY`
- `LEVEL 1 DOCTRINE_DEFINED`
- `LEVEL 2 PORTS_DEFINED`
- `LEVEL 3 SCRIPT_INTERFACE_DEFINED`
- `LEVEL 4 UTILITY_DEFINED`
- `LEVEL 5 SCRIPTED`
- `LEVEL 6 SANCTUM_VISIBLE`
- `LEVEL 7 OPERATIONAL`
