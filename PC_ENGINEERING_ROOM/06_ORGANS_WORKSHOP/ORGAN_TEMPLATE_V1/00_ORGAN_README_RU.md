# ORGAN_TEMPLATE_V1

Шаблон инженерного органа для поэтапной миграции и стандартизации рабочих контуров.

## Назначение
- Определять контракт органа.
- Принимать входы через INBOX.
- Формировать выходы через OUTBOX.
- Вести задачи, тесты, receipts и локальное состояние.

## Ограничения
- Шаблонный контур без canon authority.
- Любое повышение полномочий только через Owner approval.
