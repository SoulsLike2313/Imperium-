# ORGAN_MIGRATION_PLAN

1. Не мигрировать все 10 органов сразу.
2. Сначала создать пилотный Administratum.
3. Затем Mechanicus.
4. Затем Custodes.
5. Затем остальные органы.
6. Каждый орган должен получить contract, inbox, outbox, scripts, tasks, receipts, tests.
7. Никакой орган не получает canon/admission authority без Owner approval.
