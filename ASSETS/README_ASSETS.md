# ASSETS

ASSETS — это зона визуальных доказательств и референсов для IMPERIUM.
Здесь хранятся raw предпочтения Owner, accepted/rejected/candidate материалы и структурированные данные для будущей визуальной фабрики.

Важно:
- ASSETS не является runtime Sanctum.
- Файлы не становятся каноном автоматически.
- Канон формируется только через manifest + Owner confirmation + обновлённые visual rules.
