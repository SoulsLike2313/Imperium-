# BUNDLE ROUTE MAP V0.2

## Canonical routing
- PC Servitor bundle output: `E:\IMPERIUM_CONTEXT\LOCAL\OUTBOX\PC_SERVITOR_BUNDLES\{TASK_ID}\`
- Transport zip location: `E:\IMPERIUM_CONTEXT\LOCAL\TRANSPORT_ZIPS\` or inside the task bundle folder as transport artifact.
- Canonical repo evidence: tracked files under `E:\IMPERIUM\ORGANS\...` and other tracked repo zones.

## Evidence rule
- Folder bundle under LOCAL OUTBOX is canonical local operational evidence.
- Zip is transport only and must not be treated as canonical truth source.
- `OUTBOX` working zone inside repo is forbidden for new operational output.
