
# AGENT FACTORY FOUNDATION CONCEPT V0.1

## Purpose
- Define controlled IMPERIUM-native agent production instead of ad-hoc large-agent sprawl.

## Core Principles
- Agents are IMPERIUM-native micro-neural/task blocks.
- Not 50 huge agents consuming limits without governance.
- Shared Agent Factory foundation must govern profiles.

## Architecture Chain
Script -> Script Family -> Agent Skill -> Agent Profile -> Agent Family -> Block Wing -> Organ -> IMPERIUM Body -> Owner Brain + IMPERIUM Body = operating intelligence.

## Ownership And Governance
- Narrow profiles with explicit mission scope.
- Script family relation is mandatory.
- Agent family relation and block wing mapping are mandatory.
- Each profile has owner organ.

## Sanctum/Second Brain Observation
- Profile state, mission state, scope, gates, receipts, stop conditions, warnings must be observable.

## Allowed Scripts/Tools
- Only approved script families may be used by a profile.

## Required Gates/Receipts
- Agent factory compliance gate.
- Scope boundary gate.
- Evidence receipt gate.
- No fake green gate.

## First Recommended Agent Profiles
- Recon Cartographer.
- Gate Scribe.
- Receipt Custodian.
- Scriptorium Absorber.
- Delivery Assembler.
