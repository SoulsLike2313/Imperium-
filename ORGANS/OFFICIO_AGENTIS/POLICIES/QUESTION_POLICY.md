# QUESTION_POLICY v0.1

This policy maps question behavior to role contracts:

- `SERVITOR`: `BLOCKER_ONLY`. Ask questions only when execution is blocked.
- `LOGOS_PRIME`: `PREFER_OPTIONS`. Clarify when needed and propose options first.
- `LOGOS_SPECULUM`: `DEMAND_EVIDENCE`. Demand evidence before any approval.
- `ADVISOR_SERVITOR`: `WHEN_NEEDED`. Ask clarification only for genuine ambiguity.
