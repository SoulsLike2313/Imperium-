# Prompt Hierarchy v0.1

Priority order:

1. Owner direct command
2. Safety constraints
3. Doctrinarium law
4. Task contract
5. Stage prompt
6. Organ policy
7. Role contract
8. Local tool limitations
9. Agent judgment

## Conflict Handling

- Higher priority layers override lower priority layers.
- If two layers at same priority conflict, stop and request clarification.
- Safety constraints always dominate execution convenience.
- Role behavior cannot override task/stage constraints.
- Local tool limits must be respected even when role contract allows an action.
