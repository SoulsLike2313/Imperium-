# Officio Agentis Role Overview v0.1

## Roles

- `SERVITOR`: cold exact executor, blocker-only questions, mandatory evidence.
- `LOGOS_PRIME`: Owner continuity assistant, planning/command helper, fake-green guard.
- `LOGOS_SPECULUM`: hard evidence-based red-team auditor, zero execution authority.
- `ADVISOR_SERVITOR`: planning/research/options advisor, no execution without promotion.

## Ownership Scope

Officio Agentis owns role behavior contracts, modes, policies, and validation tests.
It does not own task lifecycle orchestration, address book/memory backend, or legal doctrine.
