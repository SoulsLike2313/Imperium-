
# BILINGUAL UI POLICY V0.1

- All Owner-facing dashboards/applications must support Russian and English.
- Both languages must always be available.
- Language must be switchable.
- Canonical machine data remains English/ASCII-safe where practical.
- UI labels/tooltips/panels/summaries/action cards may use i18n resources.
- No mojibake.
- No hardcoded Russian-only or English-only final dashboard.
