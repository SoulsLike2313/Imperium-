# Bilingual UI Foundation Gate V0.1

- gate_id: `GATE-U16-BILINGUAL-UI`
- status: `DRAFT_ACTIVE_FOR_PLANNING`

## Purpose
- Owner-facing dashboards/apps must support RU/EN from the foundation.

## Applies To
- `owner_facing_dashboards`
- `control_center_panels`
- `freelance_delivery_surfaces`

## Required Inputs
- `ui_surface_scope`
- `language_policy_ref`

## Required Evidence
- `ru_en_resource_plan`
- `language_switch_design`
- `encoding_safety_notes`

## Pass Condition
- RU and EN availability is designed/enforced for owner-facing surfaces.

## Fail Condition
- Single-language owner-facing surface is treated as final/acceptable.

## Stop Condition
- Owner-facing implementation proceeds without bilingual policy alignment.
