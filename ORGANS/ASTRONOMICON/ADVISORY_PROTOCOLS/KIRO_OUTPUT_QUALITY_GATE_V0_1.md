# KIRO OUTPUT QUALITY GATE V0.1

## Gate checks

- exact context/head is present, or missing context is explicitly labeled
- required tables are present
- paths/zones are present
- acceptance criteria are present
- checker/receipt suggestions are present
- Owner gate classification is present
- blockers/warnings are present
- verified/unverified/assumption distinctions are explicit
- no canon claims
- no fake green claims
- no unsupported execution order

## Usefulness verdict

- ACCEPT_AS_RAW_ADVISORY
- NEEDS_REWRITE
- REJECT_LOW_VALUE_ADVICE
- OWNER_DECISION_REQUIRED

## Gate rule

- Advisory output must pass this gate to be accepted as useful RAW advisory input.
- Passing this gate does not grant canon status or execution authority.
