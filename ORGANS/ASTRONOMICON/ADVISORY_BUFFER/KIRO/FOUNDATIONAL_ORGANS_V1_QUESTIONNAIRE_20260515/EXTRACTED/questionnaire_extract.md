# Foundational Organs V1 Questionnaire Extract

This advisory material is used to define what the first four foundational organs must become in V1.

Core extracted themes:

1. Each organ needs a clear role, boundaries, evidence model, and dashboard.
2. No organ dashboard may display green without real backend evidence.
3. Dashboards must read real reports, scripts, receipts, and metrics.
4. Sanctum should aggregate organ dashboards while preserving each organ's unique style.
5. The future hardening plan must be derived from Owner answers, not guessed.
6. Canonical artifacts remain English/UTF-8 safe; UI must support EN/RU.

Owner answer direction already captured in live discussion:
- Administratum owns task lifecycle execution truth and continuity/bundle collection.
- Astronomicon owns task memory, scope, stage maps, and registration workflow.
- Officio Agentis owns role/mode/agent contracts and proof of compliance with contracts.
- Doctrinarium owns law/canon/compliance and organ/task readiness gates.
- Sanctum aggregates all four as a central command bridge.
