# ASTRONOMICON PASS CRITERIA V0.1

## PASS means in this phase

- General Task universal format exists.
- Schema exists.
- Template exists.
- Basic validator exists.
- Basic parser exists.
- Basic decomposition-to-candidates script exists.
- Speculum task/stage review templates exist.
- Dashboard/workbench skeleton exists.
- Dashboard data files exist.
- No active task is falsely registered.
- Servitor future handoff path is documented.

## PASS does NOT mean

- full Astronomicon complete
- Administratum stage acceptance complete
- Speculum integration complete
- all local tasks reviewed
- registered task ready for production agent
- READY_FOR_AGENT true

## Current phase guardrails

- NO_ACTIVE_GENERAL_TASK remains truthful until explicit registration flow is implemented and used.
- Dashboard must not show fake green when active pointers are null.
