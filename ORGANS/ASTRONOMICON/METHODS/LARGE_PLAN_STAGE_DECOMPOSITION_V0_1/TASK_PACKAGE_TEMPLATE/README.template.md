# Task Package Template README

This template package is used to convert large advisory references into execution-ready task artifacts.

## Files

- `TASK_MANIFEST.template.json`: task metadata and ownership boundaries.
- `STAGE_MAP.template.json`: stage decomposition and gating.
- `SOURCE_REFERENCES.template.json`: traceability to advisory/reference sources.
- `EVIDENCE_REQUIREMENTS.template.json`: anti-fake-green evidence contract.

## Usage

1. Copy template files into a new task package directory.
2. Replace placeholder IDs, paths, and stage details.
3. Validate all JSON parses and required fields.
4. Confirm ownership boundaries and advisory/canon separation.
5. Only then prepare registration package for Astronomicon.
