# Входящие цели и желания Owner

## Как использовать

Owner бросает сюда сырые желания и будущие цели.
Агент потом анализирует эту зону и превращает в:
- Вопросы к Owner
- Кандидаты в планы (PLAN_CANDIDATES/)
- Очередь на ревью (OWNER_REVIEW_QUEUE/)
- Задачи в Task Intake

## Формат

Просто пиши. Формат свободный. Агент разберётся.
