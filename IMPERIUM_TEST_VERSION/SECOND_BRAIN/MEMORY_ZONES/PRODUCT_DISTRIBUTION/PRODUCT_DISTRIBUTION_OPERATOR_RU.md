# Оператор: Product / Distribution

## Статус: SCAFFOLD

## Что это

Зона для представления IMPERIUM как продукта — и для внутреннего анализа, и для внешней презентации.

## Два режима

### 1. OWNER_FULL_ANALYSIS (внутренний)
- Честный анализ для Owner
- Все слабости, долги, риски видны
- Никакого маркетинга
- Реальные уровни зрелости

### 2. CLIENT_STRONG_PRESENTATION (внешний)
- Сильная презентация без вранья
- Акцент на работающих возможностях
- Пробелы = пункты roadmap
- Профессиональный тон
