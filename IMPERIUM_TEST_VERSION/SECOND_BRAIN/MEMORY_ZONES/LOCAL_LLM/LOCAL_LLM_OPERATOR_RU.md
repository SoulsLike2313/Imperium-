# Оператор: Local LLM Zone

## Статус: NOT_CONFIGURED

## Что это

Зона для интеграции локальных языковых моделей на PC и Ubuntu ноутбуке.

## Принципы

- Маленький контекст — это нормально
- Медленно — это допустимо
- Перегрев — это FAIL
- Неконтролируемая фоновая работа — запрещена
- Одна модель одновременно
- Owner одобряет активацию модели

## Лимиты ресурсов

| Ресурс | Лимит |
|--------|-------|
| VRAM | 6 GB |
| RAM | 8 GB |
| CPU | 70% |
| GPU температура | 80°C |
| Контекст | 4096 токенов |
| Генерация | 2048 токенов |
| Таймаут | 5 минут |
| Фоновая работа | Запрещена |

## Что НЕ реализовано

- Реальная загрузка модели
- Реальный inference
- Реальный health check
- Автоматический мониторинг температуры
- Интеграция с Task Intake

## Следующие шаги

1. Выбрать модель (gguf формат для llama.cpp?)
2. Настроить llama.cpp или ollama
3. Создать health check скрипт
4. Подключить к Agent Ports как local_llm тип
5. Протестировать на простой задаче
