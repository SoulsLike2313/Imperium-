# Входящие комментарии Owner

## Как использовать

Owner пишет комментарий здесь или в чате.
Агент должен:
1. Создать запись в SAMPLE_OWNER_COMMENTS.json (или будущей БД)
2. Привязать к зоне памяти
3. Привязать к файлу/коммиту/задаче если применимо
4. Обновить OWNER_COMMENTS_INDEX.json

## Формат быстрого комментария

```
Дата: YYYY-MM-DD
Текст: <что Owner хочет сказать>
Привязка: <файл / задача / зона / коммит>
Действие: <что нужно сделать>
```
