# Входящие задачи

## Протокол приёма задачи

Когда задача поступает, агент должен:

1. **Принять**: "Я принял задачу"
2. **Понять**: "Я понимаю задачу как..."
3. **Найти контекст**: "Эти зоны памяти релевантны..."
4. **Задать вопросы**: "Эти вопросы нужно уточнить..."
5. **Предложить маршрут**: "Предлагаю такой путь..."
