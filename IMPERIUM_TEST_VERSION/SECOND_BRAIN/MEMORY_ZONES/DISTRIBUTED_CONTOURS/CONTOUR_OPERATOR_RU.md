# Оператор: Distributed Contours

## Статус: SCAFFOLD

## Что это

Зона для управления распределёнными контурами — машинами и средами где может выполняться работа.

## Текущие контуры

| ID | Имя | Статус | Примечание |
|----|-----|--------|-----------|
| PC-MAIN | Основной PC (Windows) | ACTIVE | Единственный верифицированный |
| VM2 | Виртуальная машина 2 | SCAFFOLD | Нет реальной проверки |
| LAPTOP-UBUNTU | Ubuntu ноутбук | NOT_VERIFIED | SSH не проверен |

## Принципы маршрутизации

1. PC-MAIN — основной контур для всей работы
2. VM2 — для изолированных задач (тесты, эксперименты)
3. LAPTOP-UBUNTU — кандидат для local LLM (когда будет настроен)

## Что НЕ реализовано

- Реальная SSH-проверка ноутбука
- Автоматическая маршрутизация задач
- Health check для VM2
- Балансировка нагрузки
- Автоматический failover

## Честность

- Не заявляем что ноутбук подключён пока нет реального SSH check
- Не заявляем что VM2 работает пока нет health receipt
- PC-MAIN — единственный контур с подтверждённым статусом
