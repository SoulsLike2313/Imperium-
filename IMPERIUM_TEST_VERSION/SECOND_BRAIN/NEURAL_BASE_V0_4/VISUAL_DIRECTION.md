# Visual Direction

## Theme
- Command-room neural chamber.
- Central luminous brain with thread-like strands and node pulses.
- Color logic is semantic, not decorative.

## Primary Palette
- Cyan: telemetry, live links, readable truth.
- Amber: warnings, owner attention, provisional states.
- Magenta: advanced modules, pending integrations, staged transitions.
- Red: blocker/failure states.
- Green: verified evidence-backed pass states.

## Motion
- Use restrained animation loops:
  - Pulse for active nodes.
  - Sweep arc for progress.
  - Glow variance for stage transitions.
- Motion must encode status, not only style.

## Node Semantics
- Task node: intake/runtime task object.
- Comment node: owner comment object.
- Memory strand: task-comment link object.
- Evidence node: receipt/report/check artifact.
- Action node: registered action contract.

## Accessibility and Honesty
- Every visual highlight must map to a concrete backend truth source.
- Never show production confidence for prototype-only states.
- Always expose manual-confirmation and disabled-action reasons.
