# Roadmap To Full Neural Operator

## Stage 0 (Completed in this task)
- Establish V0.4 base contracts, registries, prototype shell, snapshot builder, and checker.

## Stage 1
- Bind more features to module contract (Strategic Capabilities, Truth Spine, selected organs under test scope).
- Add richer evidence indexing from reports and receipts.
- Normalize status semantics across modules.

## Stage 2
- Introduce controlled action execution panel with owner-gated commands.
- Add receipt-first action logging for all enabled actions.
- Expand checker to validate action output receipts.

## Stage 3
- Integrate dynamic thread views with deterministic truth matrix bindings.
- Add node-level drill-down with direct file references.
- Add timeline overlay for task/comment/link evolution.

## Stage 4
- Add optional local model integration only after explicit local LLM verification contracts pass.
- Keep non-verified model states blocked and evidence-linked.

## Stage 5
- Production-readiness review (outside this task scope).
- Requires explicit owner approval and separate promotion policy.
