# Second Brain Neural Base V0.4

## Purpose
Second Brain Neural Base V0.4 is a test-version-only foundation for a modular neural operator environment.
It is designed so future features attach through registration contracts, not through ad-hoc one-off pages.

## Scope
- Repository scope: `IMPERIUM_TEST_VERSION` only.
- Primary area: `IMPERIUM_TEST_VERSION/SECOND_BRAIN`.
- Production claim: forbidden.

## Core Deliverables
- Architecture and contracts for module attachment.
- Registry files for features, visual tokens, truth mapping, and actions.
- Static operator prototype (`app/neural_base_v0_4.*`) with neural-shell presentation.
- Snapshot builder and checker for proof-backed status reporting.

## Foundation Status
- Status: `FOUNDATION_BASELINE_READY`.
- Local LLM and agent execution remain intentionally non-production and evidence-gated.
- Mutating actions stay disabled by default unless explicitly approved and implemented safely.

## Entry Points
- Operator prototype: `app/neural_base_v0_4.html`
- Snapshot builder: `tools/build_neural_base_snapshot_v0_4.py`
- Checker: `tools/check_neural_base_v0_4.py`
