## Принцип

Каждая зона — это папка с:
- JSON-схемой (контракт формата)
- Индексным файлом (реестр содержимого)
- Операторским гайдом на русском (как пользоваться)
- Примерами данных (MOCK/SAMPLE)

## Что НЕ реализовано

- Реальная персистентность памяти
- Реальное исполнение агентов
- Реальная интеграция с локальными LLM
- Реальная SSH-проверка контуров
- Автоматическая индексация
- Поиск по памяти

Всё это — будущие шаги после одобрения Owner.
