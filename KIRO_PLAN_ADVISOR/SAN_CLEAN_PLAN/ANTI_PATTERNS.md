## Критические запреты (БЛОКИРУЮЩИЕ)

### 1. Коммит с VM2
```

❌ ЗАПРЕЩЕНО: git push на VM2

**Почему:** VM2 — рабочая среда, не источник правды. Только PC имеет право коммитить.

### 2. Модификация Sanctum runtime
```
❌ ЗАПРЕЩЕНО: Редактировать sanctum_v0_29_qt.py

**Почему:** Sanctum — стабильный runtime. Изменения требуют отдельной арки.

### 3. Массовое удаление файлов

**Почему:** Legacy данные — часть истории. Удаление необратимо.

### 4. Изменение READY_FOR_AGENT

## Архитектурные антипаттерны

### 9. Hardcoded пути
```python
# ❌ ПЛОХО
path = "E:\\IMPERIUM_LOCAL\\bundles"

# ✅ ПРАВИЛЬНО
config = LauncherConfig()
path = config.external_context_local
```

### 10. Игнорирование платформы
```python
# ❌ ПЛОХО
subprocess.run(["ls", "-la"])  # Не работает на Windows

# ✅ ПРАВИЛЬНО
platform = detect_platform()
if platform == "WINDOWS":
    subprocess.run(["dir"])
else:
    subprocess.run(["ls", "-la"])
```

### 11. Отсутствие timeout
```python
# ❌ ПЛОХО
subprocess.run(["scp", ...])  # Может зависнуть навсегда

# ✅ ПРАВИЛЬНО
subprocess.run(["scp", ...], timeout=600)
```

### 12. Игнорирование return codes
```python
# ❌ ПЛОХО
subprocess.run(["git", "push"])
print("Done!")  # Даже если push failed

# ✅ ПРАВИЛЬНО
result = subprocess.run(["git", "push"])
if result.returncode != 0:
    raise RuntimeError("git push failed")
```
