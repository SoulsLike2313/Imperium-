# Explorer Baseline Policy V1

STATUS: READ_ONLY_VISUAL_MIRROR_BASELINE_CANDIDATE

Explorer 1.0 is a local Python/Tkinter read-only visual mirror of E:\IMPERIUM.

It may display:
- filesystem tree;
- node type;
- path;
- direct folder/file counts;
- direct markers;
- small text/json/md/py previews;
- visual helix panel;
- Copy Path;
- Open in Explorer.

It must not:
- write/edit/delete/move files;
- contact VM2;
- contact THRONE;
- run E2E;
- create watchers;
- create background automation;
- auto-repair;
- build continuity packs;
- claim readiness without receipts.

Explorer is not the source of truth.
Source of truth remains filesystem + manifests + hashes + receipts + audits + Speculum reviews + Owner decisions.
